Question-1: What are the different ways to select an element in the DOM?

There are 5 different ways to select element in Dom

  1. getElementById
  2. getElementsByClass
  3. getElementsByTagName
  4. querySelector
  5. querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent ?

  • innerHTML returns child html with all comments and everything
  • innerText returns current element innerText
  • textContent return current element innerText with hidden text
